# SEO tdpuls.com — Центр управления

## Структура

| Файл | Назначение |
|------|-----------|
| [CHANGELOG.md](CHANGELOG.md) | Лог всех SEO-изменений с датами (для корреляции со статистикой) |
| [TODO.md](TODO.md) | Активные задачи и приоритеты |
| [METRICS.md](METRICS.md) | Ключевые метрики, снимки статистики, KPI |
| [vsuzi-cluster.md](vsuzi-cluster.md) | ВСУЗИ кластер: хаб + статьи + товары |
| [content-plan.md](content-plan.md) | Контент-план: статьи, описания товаров, хабы брендов |
| [technical.md](technical.md) | Технический SEO: robots.txt, sitemap, schema, редиректы |

## Как пользоваться

1. **Перед любым SEO-изменением** — добавить задачу в TODO.md
2. **После внедрения** — перенести в CHANGELOG.md с датой
3. **Раз в неделю** — обновить METRICS.md (снять данные из Яндекс Метрики / Вебмастера)
4. **Раз в месяц** — сравнить метрики до/после ключевых изменений из CHANGELOG
