## Как снимать метрики

### Яндекс Метрика (CLI)
```bash
# Визиты за последние 7 дней
curl -s "https://api-metrika.yandex.net/stat/v1/data?ids=55564327&metrics=ym:s:visits,ym:s:users,ym:s:bounceRate&date1=7daysAgo&date2=today" \
  -H "Authorization: OAuth TOKEN"

# Органический трафик
curl -s "https://api-metrika.yandex.net/stat/v1/data?ids=55564327&metrics=ym:s:visits&dimensions=ym:s:lastTrafficSource&date1=7daysAgo&date2=today" \
  -H "Authorization: OAuth TOKEN"
```

### Яндекс Вебмастер (CLI)
```bash
# Показы и клики
curl -s "https://api.webmaster.yandex.net/v4/user/54045415/hosts/https:tdpuls.com:443/search-queries/popular?date_from=2026-02-03&date_to=2026-02-10&query_indicator=TOTAL_SHOWS&query_indicator=TOTAL_CLICKS" \
  -H "Authorization: OAuth TOKEN"
```

Токены в `docs/credentials.md`.
