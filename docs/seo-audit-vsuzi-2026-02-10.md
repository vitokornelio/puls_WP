# SEO-аудит страницы tdpuls.com/vsuzi/

**Дата:** 2026-02-10
**Статус:** Страница заблокирована от индексации (noindex/nofollow)

---

## Резюме

Страница `/vsuzi/` — качественный hub-лендинг с сильным контентом (~2000+ слов), отличной структурой и schema-разметкой. Однако **она полностью заблокирована от индексации** из-за noindex-директивы и отсутствия в sitemap. Это главная проблема, без решения которой все остальные усилия по продвижению бессмысленны.

---

## КРИТИЧЕСКИЕ ПРОБЛЕМЫ

### 1. Страница в noindex/nofollow — поисковики НЕ индексируют

**Impact:** КРИТИЧЕСКИЙ — страница невидима для поисковиков

В `functions-new.php:457` стоит:
```php
'noindex_pages' => ['vsuzi'],
```

Это добавляет `<meta name="robots" content="noindex, nofollow">` через фильтр `rank_math/frontend/robots` (строки 512-522). Страница **не может попасть в выдачу** ни Яндекса, ни Google.

**Исправление:** Убрать `'vsuzi'` из массива `noindex_pages`.

### 2. Страница НЕ в XML-sitemap

**Impact:** ВЫСОКИЙ — краулеры не узнают о странице

Проверка `sitemap_index.xml` показала 6 карт (posts, pages, products, categories) — URL `/vsuzi/` ни в одной из них. Rank Math исключает noindex-страницы автоматически.

**Исправление:** После снятия noindex страница должна автоматически появиться в `page-sitemap.xml`.

### 3. Нет canonical-тега

**Impact:** ВЫСОКИЙ — потенциальные проблемы с дублированием

Страница не содержит `<link rel="canonical">`. Rank Math обычно добавляет его, но noindex-директива блокирует и canonical.

**Исправление:** После снятия noindex Rank Math сгенерирует canonical автоматически.

---

## ПРОБЛЕМЫ СРЕДНЕГО ПРИОРИТЕТА

### 4. Нет Open Graph тегов

**Impact:** СРЕДНИЙ — плохое отображение при шаринге в соцсетях/мессенджерах

Отсутствуют `og:title`, `og:description`, `og:image`. При отправке ссылки в WhatsApp/Telegram не будет красивого превью.

**Исправление:** Добавить OG-теги через Rank Math или фильтр в `functions-new.php`. Нужен `og:image` — 1200x630px.

### 5. Нет BreadcrumbList schema

**Impact:** СРЕДНИЙ — упущенный rich snippet в выдаче

На странице есть визуальные хлебные крошки, но нет JSON-LD `BreadcrumbList`.

**Исправление:** Добавить BreadcrumbList schema в шаблон.

### 6. Большой инлайн CSS (~750 строк, ~15 КБ)

**Impact:** СРЕДНИЙ — увеличивает размер HTML, не кэшируется отдельно

**Исправление:** Вынести CSS в отдельный файл `vsuzi-hub.css`.

### 7. Нет hreflang-тегов

**Impact:** НИЗКИЙ (пока нет англ. версии)
