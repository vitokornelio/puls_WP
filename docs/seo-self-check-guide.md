# Как самостоятельно проверять продвижение tdpuls.com

> Инструкция для регулярного мониторинга SEO. Рекомендуется проверять **1 раз в неделю**.

---

## 1. Яндекс Вебмастер (главное)

**URL:** https://webmaster.yandex.ru/site/https:tdpuls.com:443/

### Что смотреть:

#### 1.1 Поисковые запросы
- Меню: **Поисковые запросы → Статистика запросов**
- Смотри график **Показы** и **Клики** за последние 30 дней
- Показы должны расти (сейчас ~600-1200/день)
- Клики должны расти (сейчас ~15-27/день)
- **CTR** — главная метрика. Если CTR растёт — значит сниппеты стали лучше
- Хорошая цель: CTR 5% для коммерческих запросов

#### 1.2 Популярные запросы
- Открой вкладку **Популярные запросы** (там же)
- Отсортируй по **Показам** — это запросы, по которым тебя чаще всего видят
- Если у запроса много показов и 0 кликов — нужно улучшать сниппет для этой страницы
- Отсортируй по **Кликам** — это запросы, которые приводят реальных посетителей

#### 1.3 Индексирование
- Меню: **Индексирование → Страницы в поиске**
- **Страниц в поиске** — должно расти (сейчас 1754)
- **Исключённых** — должно уменьшаться (сейчас 420)
- Нажми на «Исключённые» — посмотри причины исключения

#### 1.4 Диагностика
- Меню: **Диагностика сайта**
- Здесь Яндекс показывает **проблемы** — обрати внимание на «Фатальные» и «Возможные»
- Сейчас 2 возможные проблемы — проверяй, появились ли новые

#### 1.5 ИКС (Индекс качества сайта)
- Виден в сводке сайта или на https://yandex.ru/cycounter?host=tdpuls.com
- Сейчас **20** — растёт медленно, раз в месяц
- Хорошая цель: 30 через 3 месяца

## 2. Яндекс Метрика (трафик и поведение)

**URL:** https://metrika.yandex.ru/dashboard?id=55564327

### Что смотреть:

#### 2.1 Сводка
- Открой **Сводку** — общее число визитов, отказы, глубина, время
- Сравни с предыдущим периодом (кнопка «Сравнить сегменты» или выбери два периода)

#### 2.2 Источники трафика
- **Отчёты → Стандартные → Источники → Источники, сводка**
- Смотри, какой % трафика даёт **Поисковые системы** (органика)
- Сейчас: органика ~24% (1025 визитов) — должна расти
- Хорошая цель: органика >40% от общего трафика

#### 2.3 Поисковые фразы
- **Отчёты → Стандартные → Источники → Поисковые фразы**
- Здесь видно, по каким запросам люди реально приходят из поисковиков
- Обрати внимание на новые запросы, которых раньше не было

#### 2.4 Страницы входа
- **Отчёты → Стандартные → Содержание → Страницы входа**
- Покажет, на какие страницы приходят из поисковиков
- Добавь фильтр: **Источник = Поисковые системы**
- Если появились новые товарные страницы — значит SEO работает

#### 2.5 Отказы по страницам
- Тот же отчёт по страницам входа — смотри столбец **Отказы**
- Если у какой-то страницы отказы >80% — с ней проблема (плохой контент, медленная загрузка)

## 3. Google Search Console

**URL:** https://search.google.com/search-console (если подключён)

Если ещё не подключён — нужно добавить сайт и подтвердить.

### Что смотреть:
- **Эффективность** — показы, клики, CTR, позиции в Google
- **Покрытие** — сколько страниц в индексе, какие ошибки
- **Основные интернет-показатели** — Core Web Vitals (скорость)

## 4. Прямая проверка сниппетов

### В Яндексе
1. Открой **приватное окно** (Ctrl+Shift+N в Chrome)
2. Введи запрос: `site:tdpuls.com GE Vivid IQ`
3. Смотри, как выглядит сниппет — есть ли нормальный заголовок и описание?
4. Повтори для топовых запросов:
   - `phillips с-дуга купить`
   - `siemens luminos max`
   - `купить узи стационарное`
   - `incisive ct`

### В Google
1. Те же запросы, но в Google
2. Сравни сниппеты — Google обычно быстрее подхватывает изменения

**Важно:** Если сниппет всё ещё содержит `[row_inner_1]` или шорткоды — значит Яндекс ещё не переиндексировал страницу.

## 5. Проверка скорости сайта

### PageSpeed Insights
1. Открой: https://pagespeed.web.dev/
2. Введи `https://tdpuls.com`
3. Жди результат
4. **Хорошо:** зелёный (90+), **Норма:** жёлтый (50-89), **Плохо:** красный (<50)
5. Проверь отдельно для **Мобильных** и **Десктопа**

### GTmetrix
1. Открой: https://gtmetrix.com/
2. Введи URL сайта
3. Смотри: TTFB, LCP, CLS — основные метрики

## 6. Проверка Schema.org (микроразметка)

1. Открой: https://validator.schema.org/
2. Введи URL любого товара, например:
   `https://tdpuls.com/shop/tomografiya/kt/ge-healthcare-access-ct-kompyuternyj-tomograf/`
3. Должны быть:
   - **Product** — с brand, sku, offers
   - **BreadcrumbList** — хлебные крошки
   - **Organization** — данные компании
4. Если есть **ошибки** (красные) — нужно исправлять

### Альтернатива — Яндекс валидатор:
- https://webmaster.yandex.ru/tools/microtest/
- Вставь URL → проверь

### В Google:

2. Для конкретной: `site:tdpuls.com/путь-к-странице/`

## 9. Чек-лист еженедельной проверки

Распечатай или сохрани — проходи по пунктам каждый понедельник:

- [ ] **Яндекс Вебмастер:** показы/клики за неделю (выросли/упали?)
- [ ] **Яндекс Вебмастер:** новые проблемы в «Диагностике»?
- [ ] **Яндекс Вебмастер:** исключённые страницы (уменьшаются?)
- [ ] **Яндекс Метрика:** визиты за неделю vs предыдущая неделя
- [ ] **Яндекс Метрика:** % органического трафика (растёт?)
- [ ] **Яндекс Метрика:** bounce rate по страницам входа (нет ли аномалий?)
- [ ] **Сниппеты:** проверить 2-3 топовых запроса в приватном окне
- [ ] **Скорость:** раз в месяц проверить PageSpeed

## 10. Фильтрация бот-трафика

Сейчас ~45% трафика — боты из Сингапура (1 956 визитов/мес с bounce 99.5%).

### Как отфильтровать ботов в Метрике
1. Открой Яндекс Метрику → **Настройки** → **Фильтры**
2. Включи **«Фильтрация роботов»** — минимум «По строгим правилам»
3. Или создай **сегмент** с фильтром: `Регион != Сингапур` для чистых отчётов

### Как заблокировать ботов на сервере
Если нет реальных клиентов из Сингапура, можно заблокировать на уровне nginx — попросить разработчика добавить правило в конфиг.

## 11. Красные флаги (когда бить тревогу)

| Ситуация | Что делать |
|---|---|
| Показы в Яндексе резко упали (>50% за неделю) | Проверить Диагностику в Вебмастере, robots.txt, нет ли бана |
| ИКС упал | Проверить, не появились ли спам-ссылки, серверные ошибки |
| 5xx ошибки в сканировании | Проверить логи сервера (`/var/log/nginx/error.log`) |
| Страницы исчезли из индекса | Проверить robots.txt, sitemap, canonical, noindex |
| Bounce rate >90% на важных страницах | Проверить контент страницы, скорость загрузки |
| Позиции упали на 10+ пунктов | Возможно обновление алгоритма — не паниковать, следить неделю |
